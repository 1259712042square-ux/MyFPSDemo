New to GitHub?

Prerequisites:

  1. Install Git: git-scm.com/download (required — Windows, Mac, Linux)
  2. Create a GitHub account: github.com/signup

First-time setup:

  1. Click "Initialize Repository" above (if no repo exists)
  2. Create a new repository on GitHub (private recommended)
  3. Copy the repository URL and paste it in "Remote URL" above
  4. Click "Set Remote" → "Commit All" → "Push"

Daily workflow:

  • Make changes → type a message → click Commit
  • Click Push to upload to GitHub
  • Enable "Auto-commit" to auto-save before AI edits
